## Why Projects and Certifications Matter

Certifications prove you know the material. Projects prove you can *use* it. Together, they are your **portfolio** — a living resume that grows even before you finish high school.

## How to Get Project Ideas

Getting stuck on "what should I build?" is very common. Here are strategies that actually work:

### Strategy 1 — Look at Your Own Life
Ask yourself: *What is a problem I face every day that data could help solve?*

- Do you track your sports stats? → **Predict who wins a match**
- Do you read a lot? → **Book recommender system**
- Do you garden? → **Plant health classifier from photos**
- Does your family run a small business? → **Sales forecasting dashboard**

### Strategy 2 — Browse Kaggle Datasets
Go to [kaggle.com/datasets](https://www.kaggle.com/datasets) and search for topics you love. If a dataset excites you, there is a project hiding inside it.

Popular starter searches: `animals`, `sports`, `music`, `movies`, `weather`, `health`

### Strategy 3 — Ask "What Would I Google?"
Think of a question you have wondered about. For example:
- "Which movies are most similar to ones I already love?"
- "Can a computer tell if a tweet is sarcastic?"
- "What features make a song popular on Spotify?"

If you have wondered it, you can probably build a model to explore it.

### Strategy 4 — Ask an LLM (Claude, Gemini, or ChatGPT)
You have been learning how LLMs work since Week 26 — now use them as your brainstorming partner! Describe what you are interested in and ask for project ideas. Here are prompts that actually work:

- *"I am a middle school student who loves soccer. Give me 5 beginner ML project ideas using soccer data."*
- *"What datasets exist for predicting music popularity? What features would I use?"*
- *"I want to build an image classifier about animals. Walk me through the steps and what algorithm to use."*
- *"What are some surprising things you can predict with publicly available data?"*

The LLM will not build the project for you — but it is an incredible **idea generator, explainer, and debugging helper**. Think of it as a tutor available 24/7. Try Claude at [claude.ai](https://claude.ai), Gemini at [gemini.google.com](https://gemini.google.com), or ChatGPT at [chatgpt.com](https://chatgpt.com).

> **Pro tip:** If you get an idea you like, follow up with *"What dataset should I use for this?"* and *"What scikit-learn model is best for this problem?"* — you will have a full project plan in minutes.

### Strategy 5 — Start with a Dataset You Already Used
Pick any dataset from class (Iris, Titanic, MNIST, IMDB) and go deeper:
- Add a new feature
- Try a different algorithm
- Build a web UI around it
- Explain *why* the model makes each decision

### Strategy 6 — Look at Past Winning Projects
- Kaggle Competitions: [kaggle.com/competitions](https://www.kaggle.com/competitions)
- Google Science Fair: [sciencefair.withgoogle.com](https://sciencefair.withgoogle.com)
- MIT THINK Scholars: [think.mit.edu](https://think.mit.edu)

You are not copying — you are getting inspired by what is possible at your age.

## How to Host Your Project

A project that no one can see is a project that does not exist. Here is how to share yours with the world — all free!

### Option 1 — Streamlit (Easiest Web App)
**Best for:** Any ML project where someone types in input and gets a prediction

Streamlit turns your Python script into a web app in about 10 lines of code.

```python
import streamlit as st
import pickle

model = pickle.load(open("model.pkl", "rb"))

st.title("My ML App")
feature = st.number_input("Enter a value:")
if st.button("Predict"):
    result = model.predict([[feature]])
    st.write(f"Prediction: {result[0]}")
```

Run `streamlit run app.py` and you get a live web app. Then deploy to:
🔗 https://streamlit.io/cloud — free hosting, no credit card

### Option 2 — Hugging Face Spaces (Best for AI/NLP Projects)
**Best for:** Transformer models, sentiment analysis, chatbots, image classifiers

Hugging Face Spaces hosts Streamlit or Gradio apps for free. Used by researchers worldwide.
🔗 https://huggingface.co/spaces

### Option 3 — GitHub Pages (Best for Showcasing Your Work)
**Best for:** Portfolios, project write-ups, visualizations

Your GitHub repository can become a website. Add a `README.md` with your results, charts, and explanation — it automatically displays as a beautiful project page.
🔗 https://pages.github.com

### Option 4 — Google Colab (Simplest — No Setup)
**Best for:** Sharing notebooks directly

Every Colab notebook has a shareable link. You can share it with your instructor, judges, or anyone. Add markdown cells to explain what each section does.
🔗 https://colab.research.google.com

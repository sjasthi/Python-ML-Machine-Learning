üìö Training Set

The model learns from this data. Like studying from a textbook.

  • Largest portion (60-80%)
  • Model sees these examples
  • Learns patterns here

üß† Validation Set

Like practice tests. We tune the model and check progress.

  • Medium portion (10-20%)
  • Checks if we're overfitting
  • Helps adjust settings

üéØ Test Set

The final exam! Model has NEVER seen this before.

  • Smaller portion (10-20%)
  • Used only at the end
  • True measure of learning
